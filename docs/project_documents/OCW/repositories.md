## Version Control Repositories

### ocwcms

The [ocwcms GitHub repository](https://github.com/mitocw/ocwcms/) contains:

1. Custom Plone Python code: views, page templates, configuration files, and so on.
1. Engine Python scripts that are responsible for publishing the site to the webservers, mirror server, and netstorage services.
1. Static HTTP resources that are served on the HTTP (origin) servers.

### salt-ops

This repository. Contains numerous Salt states and pillar files that control how the OCW servers are configured.
